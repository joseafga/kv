# KV

[Cloudflare API](https://developers.cloudflare.com/api/resources/kv/) for KV data storage written in Crystal.

## Installation

1. Add the dependency to your `shard.yml`:

   ```yaml
   dependencies:
     kv:
       github: joseafga/kv
   ```

2. Run `shards install`
